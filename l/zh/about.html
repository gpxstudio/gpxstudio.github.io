关于 🚴

gpx.studio 是免费的 GPX 查看器和编辑器，可以做如下事情：
  • 可视化不同颜色的多条轨迹，并辅以许多不同的地图（骑行、远足、卫星等）
  • 通过添加、插入、移动或删除轨迹点来编辑轨迹
  • 反向轨迹
  • 添加或更改轨迹的时间戳
  • 查看，添加，编辑和删除航点
  • 减少航迹点的数量
  • 查看和修订文件结构
  • 缺失时自动将高程数据添加到轨迹
  • 在需要时合并多轨迹，延长时间、心率、步频、功率和温度数据
  • 从轨迹中提取轨迹段并执行任何其他操作，同时在文件中保留轨迹段的结构
  • 将文件下载到桌面或保存到谷歌云盘™，以获得可共享链接和嵌入代码。
这是在处理 GPX 文件一段时间后创建的一个单人项目。 项目源代码可在 GitHub 上找到。 在 Facebook 和 Twitter 上关注我们以获得关于工具更新的通知。

创建一个新的 GPX 文件
  1. 在顶部工具栏中，点击 新的 GPX。 底部面板上出现一个名为“新”的标签。
  2. 点击您想要通过的位置，在地图上放置锚点。
  3. 如果启用了路由，请选择适当的活动类型以便匹配最合适的道路。 禁用路由将允许您在地图上自由绘制直线。
  4. 您可以拖动点或线本身来重新规划轨迹的路由到一个新的位置。
  5. 如果你犯了错误或改变了你对行程的想法，你可以：
    • 使用撤消 和重做 按钮。
    • 点击锚点并使用 删除点 去删除它，或者简单的 Shift +点击它。

  • 点击 加载 并选择一个或多个GPX 文件加载。
  • 拖放GPX文件到窗口中的任意位置.

  • 点击 导出 然后选择是否：
    • 将所有打开的 GPX 文件合并为一个文件导出，或单独导出。
    • 包括或者不包括时间、温度、心率、步频，功率和路面数据。
    然后点击 下载 下载到您的计算机上，或 保存到谷歌云盘 以便将其保存到云端。
  • 拖放标签(底部面板下面) 到您的桌面。
共享一个 GPX 文件
gpx.studio 没有提供 GPX 文件托管。 然而，它已经丝滑的集成了谷歌云盘。 要分享 GPX 文件：
  1. 点击 导出 然后选择是否：
    • 将所有打开的 GPX 文件合并为一个文件导出，或单独导出。
    • 包括或者不包括时间、温度、心率、步频，功率和路面数据。
  2. 点击 保存到谷歌云盘
  3. 连接你的谷歌账户。
  4. 选择要保存文件的文件夹。
  5. 当文件被上传时，点击 复制链接 并发送给您的朋友！
如果您想要使用 gpx.studio 打开存储在您的谷歌云盘上的 GPX 文件，您可以安装 谷歌云盘附加组件。

内嵌 🗺

非常欢迎您使用 gpx.studio 在您的网站上显示轨迹。 有两种方法可以做到：第一种方法的文件存储在服务器上，第二种方法的文件存储在谷歌云盘。
  • 文件网址：在查询字符串的 state 参数的 urls 属性中添加指向文件的链接。 确保使用 https 对任何特殊字符进行编码，并允许 gpx.studio 请求文件，否则请求将被 CORS 策略阻止，即您应该在服务器响应中设置以下标头：

    Access-Control-Allow-Origin: *或者https://gpx.studio

  • 谷歌云盘：只需在导出时将文件保存到您的谷歌云盘并复制提供的代码即可！ 如果要显示谷歌云盘中已有的文件，请在查询字符串的 state 参数的 ids 属性中添加文件的 ID，并确保所有具有链接的用户都可以使用这些文件。

Choose the map shown by default with the parameter source among the following values:

  • outdoors: Mapbox Outdoors
  • satellite: Mapbox Satellite
  • osm: OpenStreetMap
  • otm: OpenTopoMap
  • ohm: OpenHikingMap

To use Mapbox maps, you need to specify the token parameter with a valid Mapbox token. You can also use a custom Mapbox style by directly specifying its URL with mapbox-style=<STYLE_URL>.

You can optionally add parameters to the source URL: imperial to get data in miles and feet, running to display the pace in minutes by unit of distance, distance to show distance markers, direction to show direction markers and slope to show the slope view of the elevation profile.

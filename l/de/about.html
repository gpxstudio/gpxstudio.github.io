Über 🚴

gpx.studio ist ein kostenloser Online-GPX-Viewer und Editor, der es ermöglicht:
  • Mehrere Tracks in unterschiedlichen Farben auf verschiedensten Karten (Radfahren, Wandern, Satelliten, Strava Heatmap, Overlay usw.) anzuzeigen
  • Tracks durch Hinzufügen, Einfügen, Verschieben oder Löschen von Trackpunkten zu editieren
  • Tracks umzukehren
  • Einem Track Zeitmarken hinzuzufügen
  • Wegpunkte anzuzeigen, hinzuzufügen, sie zu bearbeiten und zu löschen
  • Die Menge an Trackpunkten zu reduzieren
  • Fehlende Höhendaten automatisch hinzuzufügen
  • Tracks zusammenzuführen und bei Bedarf um Zeit-, Herzfrequenz-, Trittfrequenz-, Leistungs- und Temperaturdaten zu ergänzen
  • Segmente aus Tracks zu extrahieren und viele andere Aktionen zur Bearbeitung der Segmentstruktur in Dateien anzuwenden
  • Daten lokal zu speichern, oder auf Google Drive™ um einen Link zum Teilen und Einbettungscode zu erhalten
Es ist ein Ein-Personen-Nebenprojekt, das entstand, nachdem ich einige Zeit mit GPX-Dateien herumgespielt habe. Der Quellcode des Projekts ist öffentlich auf Github verfügbar. Folgen Sie uns auf Facebook und Twitter, um über Updates des Tools auf dem Laufenden zu bleiben.

Mapbox Community 🤝

Mapbox ist das Unternehmen, das einige der schönen Karten auf dieser Website zur Verfügung stellt. Wir verlassen uns auch auf ihre Dienste, um Routen zwischen Punkten die Sie hinzufügen oder verschieben abzurufen, um den Tracks genau Höhendaten hinzuzufügen, oder Sie an den richtigen Ort zu bringen, wenn Sie in der Suchleiste suchen.

Sie haben ein Programm namens Mapbox Community ins Leben gerufen, um gemeinnützige Organisationen, Bildungseinrichtungen und Positive-Impact Organisationen zu unterstützen (siehe die offizielle Website auf der sie einige ihrer Kooperationen vorstellen). Wir sind sehr glücklich diesem Programm beigetreten zu sein und von einem großen Rabatt auf alle API-Preise zu profitieren. Dank ihnen dürfte die Finanzierung dieses Projekts einzig über Spenden auch langfristig möglich sein..

Ich bin (und alle Benutzer der Website können es sein) sehr dankbar für ihre Unterstützung und Erreichbarkeit.

Helfen Sie dabei, die Website kostenfrei (und werbefrei) zu halten 🙏

Bei jedem Hinzufügen oder Verschieben eines Wegpunktes führen wir eine Anfrage an Mapbox Directions API durch um eine Route im Straßennetz zu erhalten. Zusätzlich fragen wir exakte Höhenangaben zu den neuen Wegpunkten von Mapbox Terrain-RGB tiles an..

Bedauerlicherweise ist das sogar mit der Unterstützung von Mapbox (siehe Mapbox Community oben) nicht kostenfrei. Weitere abrechenbare Vorgänge sind Kartenladevorgänge und Geokodierungsanfragen (Suche nach einem Ort in der Suchleiste).

Erstellen einer neuen GPX-Datei
  1. Klicken Sie in der oberen Symbolleiste auf Neue GPX. A new tab called "new" appears below the bottom panel.
  2. Place anchor points on the map by clicking at the locations you want to go through.
  3. In the settings , two options control how two consecutive points are connected:
    • use to select the correct activity and use the most appropriate roads.
    • use to alternate between routing (following real paths) or "as-the-crow-flies" (straight lines between anchor points).
  4. You can drag a point or the line itself to re-route the trace through a new location.
  5. If you make a mistake or change your mind about the itinerary, you can:
    • use the undo and redo buttons.
    • right-click on an anchor point to remove it.

Teile eine GPX Datei
gpx.studio does not provide GPX file hosting. However, it integrates smoothly with Google Drive. To share GPX files:
  1. Click on Export and then choose whether to:
    • merge all open GPX files or export them separately.
    • include time, temperature, heartrate, cadence and power data or not.
  2. Click on Save to Google Drive.
  3. Connect to your Google account.
  4. Select the folder where you want to save the files.
  5. When the files are uploaded, click on Copy link and send it to your friends!
If you want to use gpx.studio to open GPX files stored on your Google Drive, you can install the Google Drive Add-on.

  • drag an anchor point or the line itself to re-route the trace through a new location.
  • right-click on an anchor point to remove it.
  • right-click on an anchor point or on the line to split the trace at a chosen location.
If you make a mistake or change your mind about the itinerary, use the undo and redo buttons.

  • use to select the correct activity and use the most appropriate roads.

If you edit a GPX file containing time, temperature, heartrate, cadence or power data, new track points will automatically contain the average value for each of those fields.

Füge mehrere GPX Dateien zusammen
To merge two GPX files:
  1. Below the bottom panel, select the tab of the first GPX file you want to merge.
  2. Click on (available only when more than two files are open).
  3. Click on the tab of the second GPX file to merge. The track points of the second GPX file will come after those of the first GPX file.
If you want to merge all open GPX files at once:
  1. Below the bottom panel, put the tabs in the order you want them to appear in the merged GPX file.
  2. Click on Export.
  3. Check the box Merge all traces.
  4. Click on Download to download the files on your computer or on Save to Google Drive to save it in the cloud.
When merging multiple GPX files: missing time, temperature, heartrate, cadence or power data will be automatically extended with the average value across merged files. If some GPX files contains timestamps, it is ensured that the timestamps of the merged file are strictly increasing (the timestamps of some files are shifted if necessary).

Embedding 🗺

You are very welcome to display traces on your website with gpx.studio. There are two ways to do it: the first one with files stored on a server and the second one with Google Drive.
  • File URL: Add the link to the file in the urls attribute of the state parameter of the query string. Make sure to use https, to encode any special characters and that gpx.studio is allowed to request the files, otherwise the requests will be blocked by the CORS policy, i.e. you should set the following header in your server response:

    Access-Control-Allow-Origin: * or https://gpx.studio

  • Google Drive: Just save the files to your Google Drive when exporting and copy the code provided! If you want to display files that you already have on your Google Drive, add the ID of the file in the ids attribute of the state parameter of the query string and make sure they are available to all users with the link.

Choose the map shown by default with the parameter source among the following values:

  • outdoors: Mapbox Outdoors
  • satellite: Mapbox Satellite
  • osm: OpenStreetMap
  • otm: OpenTopoMap
  • ohm: OpenHikingMap
  • cosm: CyclOSM
  • ign: IGN (FR)

To use Mapbox maps, you need to specify the token parameter with a valid Mapbox token.

You can optionally add parameters to the source URL: imperial to get data in miles and feet, running to display the pace in minutes by unit of distance, distance to show distance markers and direction to show direction markers. Google Drive

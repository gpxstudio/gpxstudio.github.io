About

gpx.studio is a free online GPX viewer and editor which allows to :

  • visualize multiple traces in different colors and aided by many different maps (cycling, hiking, satellite, Strava heatmap overlay, etc)
  • edit traces by adding, inserting, moving or deleting points
  • reverse traces
  • add timestamps to a trace
  • view, add, edit and remove waypoints
  • automatically add elevation data to traces if missing
  • merge multiple traces or export them separately, extending time, heart rate, cadence and temperature data where needed
  • save the result on your computer or to your Google Drive to get a shareable link and embedding code

It is a one-person side project created after messing with GPX files for some time. If you like the tool and use it frequently, please consider donating as to cover bills from API calls (map display, routing and elevation features) and encourage me to keep improving the website. Thank you for your support!

The project source code is available on GitHub where I also put future developments. Follow gpx.studio on Facebook and Instagram to be notified about updates, read and share stories.
